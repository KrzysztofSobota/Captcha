My another idea is game based on common used in the Internet security - "captcha". My game is case-sensitive version, but what else is so special here?

You can choose from any of 5 difficulty levels of this "captcha". Name of recently selected level (short text message) are showed inside the box in the right side of the stars. Description what "surprise" is included in every level you can find in the table below.

You can start game instantly in "easy" (default) level or click any of the non-yellow star. If you made a choice and decide to change the level, refresh the page (F5) or click yellow reset button near the captcha code. This will change the game level and the code.

I believe that you can find something perfect just for you. Not too easy and not too hard.

So go on and try captcha in a one of your favourite levels!
